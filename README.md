# Azure App CI/CD Project

## Overview
This project demonstrates deploying a Node.js application to Azure App Service with a CI/CD pipeline using GitHub Actions.

## Architecture
- Node.js + Express application
- Azure App Service (Linux)
- GitHub Actions CI/CD pipeline

## Technologies Used
- Node.js
- Express.js
- npm
- Git & GitHub
- GitHub Actions
- Azure App Service
- Azure CLI

## Features
- Automated deployment on push to main branch
- Cloud-hosted web application
- CI/CD pipeline using GitHub Actions

## How It Works
1. Developer pushes code to GitHub
2. GitHub Actions workflow is triggered
3. Pipeline installs dependencies and packages the app
4. App is deployed to Azure App Service
5. Changes go live automatically

## Learning Outcomes
- Built CI/CD pipeline for cloud deployment
- Deployed Node.js application to Azure
- Configured secure authentication using service principal
- Troubleshot real-world deployment issues
